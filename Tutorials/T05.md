   a. Flexible?  
   b. Easy (shorter development time)?  
   c. Adaptive / Dynamic?

Provide ONE(1) justification for each of your answers in (a) to (c) above.

3. Debate over the use of nested layout in your mobile app UI.

Advantages:
- Nested layout make design easier to organize
- Improve readability of the layout file

Disadvantages:
- Too many nested layout reduces performance, the system require
more time to render
- UI is harder to maintain due to the complexity of the program
