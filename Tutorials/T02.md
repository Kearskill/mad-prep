**Tutorial 2: Introduction to Mobile App Development**

1. Complete the following table for Android and iOS:

| Native App  | Source Code Extension/ Scripting Language | Compiler                           | Distributable Package                 | Publishing Store  |
|-------------|-------------------------------------------|------------------------------------|---------------------------------------|-------------------|
| Android App | `.kt` (Kotlin) or `.java` (Java)          | D8 / R8 (converts bytecode to DEX) | `.apk` or `.aab` (Android App Bundle) | Google Play Store |
| iOS App     | `.swift` (Swift) or `.m` (Objective-C)    | LLVM / Clang                       | `.ipa` (iOS App Store Package)        | Apple App Store   |


2. Complete the following table for the comparison among three types of mobile apps:

| **Characteristics**             | **Native Apps**                                                               | **Hybrid Apps**                                                               | **Web Apps / PWA**                                            |
|---------------------------------|-------------------------------------------------------------------------------|-------------------------------------------------------------------------------|---------------------------------------------------------------|
| Platform                        | Platform-specific (Android/iOS)                                               | `Cross-platform`                                                              | `Any browser`                                                 |
| Language                        | `Kotlin, Java`                                                                | `HTML, JS, CSS`                                                               | HTML, JS, CSS                                                 |
| Performance                     | Best                                                                          | `Good`                                                                        | Moderate                                                      |
| Development cost                | High                                                                          | `Moderate`                                                                    | `Low`                                                         |
| User experience (UX)            | `Excellent`                                                                   | `Good`                                                                        | `Fair`                                                        |
| Device Access                   | `Full`                                                                        | `Partial (depends on framework and plugins)`                                  | Limited                                                       |
| Offline support                 | `Yes                                                                          | Yes                                                                           | `Limited (depends on caching and PWA support)`                |
| Update method                   | `Through App Store`                                                           | `Through App Store`                                                           | `Instant (via server update)`                                 |
| Advantages (At least 2 each)    | `High performance and speed` and `Full access to device features`             | `Single codebase for multiple platforms` and `Faster development than native` | `Easy to maintain and update` and `No installation required`  |
| Disadvantages (At least 2 each) | `Expensive and time-consuming` and `Requires separate code for each platform` | `Slightly slower than native` and `Limited access to some native features`    | `Lower performance` and `Limited offline and hardware access` |
| Best for                        | `Performance-critical apps, gaming`                                           | App needing cross-platform capability                                         | `Content-based apps or simple tools`                          |
| Examples (At least 3 each)      | `Spotify, Adobe Photoshop, Angry Bird`                                        | `Discord, Microsoft Teams, WhatsApp`                                          | `Pinterest (PWA), Canva, Figma`                               |
